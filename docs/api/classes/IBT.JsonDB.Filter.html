\IBT\JsonDB

\IBT\JsonDBFilter

Filter class enables the selection of a subset of JSON documents.

The Filter class is acts as a query builder to enable you to search and retrieve a subset of JSON documents. It uses similar/familiar SQL constructs such as 'Where', 'Order By' etc...

$c = new Client();
$c->setup();
$col = $c->newCollection("users");
$col->insert('{"name":"jason bourne", "category":"agent"}');
$f = $col->newFilter();
$f->where('category', 'in', ['agent'])->run();
