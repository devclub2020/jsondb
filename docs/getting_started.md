# Getting started

## Installation (new project)

```
composer init --require="johnwilson/jsondb"
```

## Call autoloader

```PHP
require_once __DIR__ . '/vendor/autoload.php';
```

## Import the relevant namespaces as required:

```PHP
use IBT\JsonDB\Client;
use IBT\JsonDB\Collection;
```
